# Zulcan Crew

A living 13-member archetypal multi-agent system based on the Tzolkin calendar (Ian Lungold framework).

## Overview

The Zulcan Crew is a council of 13 distinct agents, each with deep, stable soul files. A Master Router intelligently decides which members collaborate on any given task.
